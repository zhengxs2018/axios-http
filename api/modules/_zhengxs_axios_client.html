AxiosRetryConfig

AxiosRetryConfig: { retries?: number; shouldResetTimeout?: boolean; retryCondition?: any; retryDelay?: any }

Type declaration

  • Optional retries?: number

    The number of times to retry before failing default: 3

  • Optional shouldResetTimeout?: boolean

    Defines if the timeout should be reset between retries default: false

  • retryCondition?:function
    • retryCondition(error: AxiosError<any>): boolean | Promise<boolean>

    • A callback to further control if a request should be retried. By default, it retries if the result did not have a response.

      Parameters

      • error: AxiosError<any>

      Returns boolean | Promise<boolean>

  • retryDelay?:function
    • retryDelay(retryCount: number, error: AxiosError<any>): number

    • A callback to further control the delay between retry requests. By default there is no delay.

      Parameters

      • retryCount: number
      • error: AxiosError<any>

      Returns number
